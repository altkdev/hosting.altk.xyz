About Products

Full Build

The "Full Build" package includes a fully writen website, made exactly to do what you would like it to do. It also includes a subdomain, and unlimited free custom URLs for all that you need.


Subdomain

The "Subdomain" package is a "bring your own site" kind of deal. This includes the subdomain (i.e. "example.altk.xyz") and unlimited free custom URLs.


Custom Domain

The "Custom Domain" package includes you own personalized domain with DNS management. This package also includes Email hosting using the Zoho emailing service. this is best paired with the "Full Build" package to get the most out of your new site.


Hosting

Basic hosting package. Must be paired with other package


Custom URL

This option is the second most basic option. This package includes only one custom URL (i.e. "altk.xyz/example"). YOU DO NOT NEED THIS IF YOU HAVE "Subdomain", "Custom Domain", OR "Full Build".
